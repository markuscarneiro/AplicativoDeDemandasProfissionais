# Imagens de Fundo - Login

## Como substituir a imagem de fundo

### Localização do arquivo:
```
demandas/static/demandas/images/background-login.jpg
```

### Instruções:
1. Substitua o arquivo `background-login.jpg` pela sua imagem personalizada
2. Mantenha o mesmo nome do arquivo ou atualize a referência no CSS
3. Formatos aceitos: jpg, png, webp
4. Resolução recomendada: 1920x1080 ou superior
5. Tamanho máximo recomendado: 2MB para melhor performance

### Características recomendadas para a imagem:
- **Resolução**: Mínimo 1920x1080, ideal 2560x1440
- **Formato**: JPG (melhor compressão) ou PNG (melhor qualidade)
- **Estilo**: Imagens com áreas menos detalhadas no centro (onde fica o formulário)
- **Contraste**: Preferencialmente imagens mais escuras ou com contraste suave
- **Conteúdo**: Imagens profissionais, paisagens, abstratas ou relacionadas ao negócio

### Exemplos de imagens adequadas:
- Paisagens urbanas noturnas
- Escritórios modernos
- Padrões geométricos suaves
- Imagens corporativas com tons azuis/cinzas
- Abstratos com gradientes suaves

### Dica de otimização:
Para reduzir o tamanho do arquivo, use ferramentas como:
- [TinyPNG](https://tinypng.com/) para PNG
- [JPEGmini](https://www.jpegmini.com/) para JPG
- [Squoosh](https://squoosh.app/) - ferramenta online gratuita do Google
